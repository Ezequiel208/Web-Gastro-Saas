# 🚀 Modelo Gastro SaaS - Starter Kit

¡Buenas! Soy **Ezequiel Paez**. Bienvenidos a uno de mis primeros proyectos liberados para la comunidad. Este es un **Starter Kit de alto impacto** diseñado para modelos de negocio SaaS, específicamente orientado al nicho gastronómico, pero 100% adaptable a cualquier rubro que necesite una landing pro.

> "Libero este código porque creo que el conocimiento se comparte. La idea es que si estás arrancando o necesitás una base sólida para un cliente, no tengas que renegar con la estructura inicial y puedas enfocarte en lo que importa: la lógica de tu negocio."

---

## ✨ ¿Por qué este proyecto?
Este proyecto es una versión evolucionada de mis primeras exploraciones en landing pages de alto rendimiento.

* **Intención:** Ofrecer una plantilla profesional con estética premium (dark mode, animaciones fluidas) para que la uses de base, la rompas toda y la adaptes a tu gusto.
* **Herramientas Modernas:** Como desarrollador, utilizo IA como herramienta para optimizar tiempos, automatizar tareas repetitivas y consultar mejores prácticas, demostrando que la tecnología es nuestra mejor aliada para entregar calidad en tiempo récord.

---

## 🛠️ Tech Stack

Para que esto vuele, usé lo mejor del ecosistema React:

* **Framework:** `React 19` + `TypeScript` (Tipado fuerte para evitar errores).
* **Build Tool:** `Vite` (Entorno de desarrollo instantáneo).
* **Styling:** `Tailwind CSS` (Estilos rápidos, modernos y optimizados).
* **Animations:** `GSAP` + `ScrollTrigger` (Animaciones de alto nivel).
* **Smooth Scroll:** `Lenis` (Scroll suave para ese toque premium).
* **Icons:** `Lucide React` (Iconografía limpia y consistente).

---

## ⚙️ Configuración
Casi todo lo visual y los textos están centralizados en el archivo: `src/data/config.ts`. 

Si querés cambiar el nombre del proyecto, los precios o las funcionalidades, **tocás ahí y se actualiza toda la web**. No tenés que andar navegando entre componentes para cambiar un texto.
